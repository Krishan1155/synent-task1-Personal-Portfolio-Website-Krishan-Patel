# Personal Portfolio Website

## Overview
This is my personal portfolio website built using HTML, CSS and JavaScript. The website showcases my profile, skills, projects, experience, and contact information in a modern and responsive design.

## Features
- Responsive Design
- Modern UI/UX.
- About Me Section
- Skills Section
- Projects Showcase
- Experience Timeline
- Contact Form
- Social Media Links
- Smooth Scrolling Navigation

## Technologies Used
- HTML5
- CSS3
- JavaScript
- Tailwind CSS
- Font Awesome

## Installation

1. Clone the repository
git clone https://github.com/yourusername/personal-portfolio.git

2. Open the project folder
cd personal-portfolio

3. Open index.html in your browser.
